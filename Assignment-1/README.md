# Assignment 1 - Identity Echo Interface

## Objective
Build a simple interactive web application using Streamlit that collects user input, validates it, and displays appropriate responses.

## Features
- User-friendly Streamlit interface
- Name input field
- Message input field
- "Transmit" button to submit data
- Input validation using conditional statements
- Error message for empty name
- Warning message for empty message
- Success message for valid input
- Token Cost Estimator (Optional Challenge)

## Technologies Used
- Python
- Streamlit

## How to Run

1. Install Streamlit (if not already installed):

```bash
pip install streamlit
```

2. Navigate to the project directory.

3. Run the application:

```bash
streamlit run app.py
```

4. Open the local URL displayed in your browser.

## Learning Outcomes
- Building interactive web applications with Streamlit
- Collecting user input
- Using buttons to trigger actions
- Implementing conditional logic (`if`, `elif`, `else`)
- Displaying different Streamlit alert components
- Understanding basic AI token estimation
