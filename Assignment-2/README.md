# Assignment 2 - Upgrading the AI Multiverse

## Objective

Upgrade a basic Gemini AI chatbot into a more interactive and visually appealing chat application using Streamlit.

## Features

- Gemini AI integration
- Sidebar with application settings
- Multiple AI personalities
- Adjustable personality intensity slider
- Chat-style interface using `st.chat_message()`
- Dynamic emoji avatars based on selected personality

## Technologies Used

- Python
- Streamlit
- Google Gemini API
- python-dotenv

## How to Run

1. Install the required packages:

```bash
pip install streamlit google-generativeai python-dotenv
```

2. Create a `.env` file and add your Gemini API key:

```env
GEMINI_API_KEY=your_api_key_here
```

3. Run the application:

```bash
streamlit run app.py
```

## Learning Outcomes

- Streamlit sidebar components
- Prompt engineering
- Gemini API integration
- Dynamic UI with chat components
- Using conditional logic for avatars
